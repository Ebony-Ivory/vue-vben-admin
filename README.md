## Install and use

- Get the project code

```bash
WxP
git clone http://174.35.64.98/yuke.lu/vben-base2.git

C7N
git clone http://code.c7n.uaes.com/tf-wxp-portal/wxp-template-vue.git
```

- Installation dependencies

```bash
cd {folder_name}

pnpm install

```

- run

```bash
pnpm dev
```

- build

```bash
pnpm build
```
